1. What is Semantic HTML? 
    Semantic HTML is a way of writing HTML documents using tags that describe the information they contain.

2. What is HTML used for? 
    HTML is used to create web pages, specifically the structure and content.

3. What is an attribute and where do we put it? 
    An attribute is additional information about an HTML element that is placed in the leading tag of the element.

4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is used to display the main title of a webpage. It should only be used once on a page.

5. Name two tags that have required attributes
    Both the img tag and a tag have required attributes.

6. What do we put in the head of our HTML document? 
    The head of our HTML documents should contain the ftitle, style, meta, link, script, and base elements.

7. What is an id? 
    An id is an attribute that is used to declare a specific identity to an element.

8. What elements can I add an id to? 
    An id can be used on any HTML element.

9. How many times can I use the same id on a page? 
    An id can be referenced numerous times on a page, but it can only be declared to reference an element once.

10. What is a class? 
    A class is an attribute used to declare an identity to an element or group of elements.

11. What elements can I add a class to? 
    A class can be added to any HTML element.

12. How many times can I use the same class on a page? 
    A class can be used as many times as is neccessary.

13. How do I get my link to open in a new tab?
    Using an a tag with the attribute target="_blank" will open a link in a new tab.

14. What is the alt attribute in the image tag used for? 
    The alt attribute in the image tag is used to provide an alternative option of understanding the image for those with a visual impairment. A screen reader can be used to read an alt attribute.

15. How do I reference an id?
    An id can be referenced using #.

16. What is the difference between a section and a div
    A section is semantic and provides information whereas a div does not.

    A parent child selector is used to select all elements that are a child of the specific parent element. This would be useful if you were trying to select list items that are a direct child of an unordered list and not any other list items nested in a deeper unordered list.
